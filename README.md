# Flavordle
A higher or lower game for Flavortown, guess if projects have more or less time spent! 

Get as many right and get the best high score!

This project was written with SvelteKit and Tailwind CSS.

# Demo
I have hosted a demo on Netlify [here](https://flavordle.netlify.app) so you don't need to set up anything!

# Developmemt
To setup a local dev server, enter the commands:
```
bun install
bun run dev
```

To build the project, use the command
```
bun run build
```
